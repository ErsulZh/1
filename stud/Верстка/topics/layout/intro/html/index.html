Введение в HTML

HTML — это язык разметки для создания веб-страниц. Он определяет структуру документа с помощью тегов.

Пример HTML-документа

<!DOCTYPE html>
<html>
  <head>
    <title>Пример</title>
  </head>
  <body>
    <h1>Заголовок</h1>
    <p>Текст абзаца</p>
  </body>
</html>

Задание

Создайте свою первую HTML-страницу с заголовком и абзацем. Попробуйте изменить текст и посмотреть результат в браузере.
