Основы CSS

Введение в CSS

CSS — это язык стилей для оформления веб-страниц. Он позволяет задавать цвета, шрифты, отступы и многое другое.

Пример CSS

body {
  background: #fff;
  color: #222;
}
h1 {
  color: #174ea6;
}

Задание

Добавьте CSS-стили к вашей HTML-странице: измените цвет текста, задайте фон и поэкспериментируйте с отступами.
